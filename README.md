# Sistema de Gestão
Este é um sistema de gerenciamento desenvolvido em Visual Basic utilizando Windows Form e banco de dados SQL Server que permite gerenciar dados de clientes e produtos.

## Pré-requisitos
Antes de executar o aplicativo, certifique-se de que você tenha os seguintes pré-requisitos instalados:

- Visual Studio
- SQL Server
- .NET Framework
- Windows Form
- Guna UI

## Recursos
O aplicativo possui as seguintes funcionalidades:

- Cadastro de Clientes: Registre informações de clientes, como nome de Usuário, Senha, Email, Data de Cadastro, etc.
- Cadastro de Produtos: Registre informações sobre os produtos, incluindo nome do produto, Preço, Data de Cadastro, Observação, etc.
